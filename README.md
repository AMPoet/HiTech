## 🛠 Installation & Setup  

1️⃣ Clone the repository:  
```bash
git clone https://github.com/AMPoet/HiTech.git
```
2️⃣ Navigate to the project folder:  
```bash
cd HiTech/Codes
```
3️⃣ Install dependencies (if required):  
```bash
pip install -r requirements.txt
```
4️⃣ Run the Python script:  
```bash
python script.py
```
